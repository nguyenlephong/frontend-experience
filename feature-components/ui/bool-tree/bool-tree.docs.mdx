# Introducing the `BoolTree` component

The `BoolTree` component is a versatile UI component that can be used in a variety of web applications. It offers the following features:

- Add a and/or condition
- Delete a condition block
- Duplicate a condition block
- Custom your logic on condition block via metaData props
- Allow hide/show data structure

## Installation

To install the `BoolTree` component, you can use the following command:
```bash
npm i @nguyenlephong/feature-components.ui.bool-tree
```

```jsx
import React from 'react';
import { BoolTree } from './bool-tree';

function MyComponent() {
  return (
    <BoolTree />
  );
}
```
